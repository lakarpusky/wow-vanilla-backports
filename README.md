# wow-vanilla-backports

A collection of WoW addons backported to the **1.12.1 vanilla client** for use on [TurtleWoW](https://turtlecraft.gg/) and other private servers.

Every addon here is written in strict Lua 5.0 with no external dependencies — no libraries, no SuperWoW required.

## Addons

| Addon | Original Version | Description |
|---|---|---|
| [SpellReplay](SpellReplay/) | WotLK 3.3.5a | Scrolling spell icon strip — shows your casted spells in order with damage, heals, resists, and auto-attacks |

## Install

1. Download or clone this repo
2. Copy the addon folder (e.g. `SpellReplay/`) into your `Interface\AddOns\` directory
3. Make sure the `.toc` file is directly inside: `Interface\AddOns\SpellReplay\SpellReplay.toc`
4. Restart the game or `/reload`
